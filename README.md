# ロボトレース コースレイアウト補助ツール

ロボトレース用のコースを容易に作成するためのWebアプリケーション。

## 現在の実装機能

### 盤面表示機能
- 1枚の板サイズ: 900mm × 1350mm
- 横と縦の枚数を指定して複数の板を配置可能
- 入力欄で枚数を指定し、「更新」ボタンで盤面を更新
- 画面に収まるように自動的に拡大縮小
- スクロールなしで全体表示

### 座標グリッド表示
- 100mm間隔で薄い色のグリッド線を表示
- 左下を(0,0)とした座標系
- X座標とY座標のラベル表示

### STARTポイント配置機能
- X座標とY座標を入力してSTARTポイントを配置
- 赤い丸で表示
- 任意の位置に配置可能

### GOALポイント配置機能
- STARTから1000mm離れた位置に配置
- X軸またはY軸に平行な方向のみ選択可能
- プルダウンメニューで方向を選択（右側、左側、上側、下側）
- 座標が負の値になる方向は選択肢に表示されない
- 青い丸で表示

## 使用方法

1. `index.html` をブラウザで開く
2. 「横」と「縦」の入力欄に板の枚数を入力（自動で盤面が更新されます）
3. START X、START Y に座標を入力（自動でSTARTポイントが表示されます）
4. プルダウンメニューからGOALの方向を選択（自動でGOALポイントが表示されます）

## ファイル構成

- `index.html` - メインHTML
- `style.css` - スタイルシート
- `script.js` - JavaScript（盤面描画ロジック）
- `README.md` - このファイル
