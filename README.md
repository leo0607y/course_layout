# ロボトレース コースレイアウト補助ツール

ロボトレース用のコースを容易に作成するためのWebアプリケーション。

## 現在の実装機能

### 盤面表示機能
- 1枚の板サイズ: 900mm × 1350mm
- 横と縦の枚数を指定して複数の板を配置可能
- 入力欄で枚数を指定し、「更新」ボタンで盤面を更新
- 画面に収まるように自動的に拡大縮小
- スクロールなしで全体表示

## 使用方法

1. `index.html` をブラウザで開く
2. 「横」と「縦」の入力欄に板の枚数を入力
3. 「更新」ボタンをクリックして盤面を表示

## ファイル構成

- `index.html` - メインHTML
- `style.css` - スタイルシート
- `script.js` - JavaScript（盤面描画ロジック）
